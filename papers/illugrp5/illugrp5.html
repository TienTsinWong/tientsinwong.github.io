Real-Time Relighting on Commodity PCs

Tien-Tsin Wong, Zhouchen Lin, Heung-Yeung Shum and Chi-Wing Fu
Submitted to Journal of Graphics Tools

Abstract

In this paper, we propose a real-time and simple relighting method for image-based computer graphics by utilizing widely-used commodity texture-mapping hardware on ordinary PCs. Hence, real-time relighting for web applications becomes possible without purchasing specialized hardware. We introduce the reflected irradiance field, as a dual of the light field, to characterize illumination by using the two-plane parameterization model. This formulation not only introduces more freedom for controlling light source positions, but also transforms the relighting problem to a texture-mapping process. To relight an image-based scenery, we warp the texture-mapped patches. Based on the superposition principle, we can relight these intensity-modulated texture-mapped patches for each light source and cache the resultant relighted images. Illumination due to multiple light sources can be efficiently achieved by blending these images and real-time relighting is evidenced to be possible on commodity graphics hardware, such as the Nvidia GeForce3. To demonstrate its feasibility, experiments on both synthetic and real-world objects are conducted. For capturing real-world scenes, a computer-controllable system is built so as to precisely sample the reflected irradiance field.

Movies

The following movies demonstrate the relighting result of both real and synthetic object. All movies are captured by DV in real-time, hence they demonstrate the real-time relighting ability.
Toy Bride (real): A toy bride lowered in a partially filled glass. It demonstrates the ability in representing refraction and caustics effects. (Size: 2.1MB, MPEG1)
Teapot (synthetic): In this movie, the image-based teapot is relit in real-time by multiple light sources, each with different color. (Size: 6.5MB, MPEG1)
Bucky Ball (synthetic): This examples demonstrates the ability in representing shadow as well as highlight. (Size: 3.3MB, MPEG1)
Furry Dog (real): The last movie shows the relighting of a real furry toy dog. It demonstrates the ability in capturing reflection from complex structure. (Size: 2.4MB, MPEG1)
