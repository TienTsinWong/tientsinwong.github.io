Introduction

Arti reduces the ringing artifacts exhibited in JPEG images. Such ringing artifacts are annoying. It is especially useful for retouching cartoon (anime) images. To our knowledge, it is the fastest tool available in removing ringing artifacts. Simply try it yourself.

System Requirement

Hardware: Pentium III or above, 128MB RAM, installed with graphics boards listed below
OS: Windows 98/NT/2000/XP, with OpenGL

Installation Instruction

  1. Download arti 1.30
  2. Unzip the arti.1.30.zip into a subdirectory, say arti
  3. Simply run arti.exe

User Manual

Basic Steps:
  1. Press "Load input" - Load the image to process (JPEG)
  2. Press "Clean" - Clean the ringing artifacts
  3. Press "Save output" - Save the cleaned image (BMP)
